c++ learning notes 1

1.main函数
int main()
{
statements;
return 0;
}
2.声明不分配内存空间，定义时才分配
3.可以使用sizeof函数查看类型占用字节，eg.sizeof(int);可以用库文件来查看编译器支持的数值范围，其包含的常量可直接输出，eg.INT_MAX,SHORT_MAX,LONG_MAX,etc
4.关于cout对象的函数
cout.put输出单字符
5.可使用八进制或十六进制或Unicode码来表示字符，以\开头
6.宽字符类型wchar_t,不过要用L前缀来表示，eg.wchar_t w=L’P’;
7.cout在输出数字时可以设置格式cout.setf(ios_base::fixed)以定点输出cout.setf(ios_base::showpoint)以小数格式输出，cout.precision(num)设置小数精度
7.关于const
const要声明时就给出定义，eg.const a=10;
8.c++中强制类型转换可以用(type)var或者type(var)
9.字符串数组的使用
eg.char s[2]={‘a’,’\0’}//这种形式必须要有’\0’，否则就不是字符串数组，当使用字符串处理函数时可能出错
eg.char s[3]=”ab”;
eg.char s[]=”hello”;//以上两种，’\0’会自动的加上
10.关于cin对象的函数
cin会保留换行符，eg.cin>>a;当输入数据并回车，回车会留在缓冲区，必要时需用cin.get消化掉
cin使用空白来界定字符串，eg.Hello c++将被当做两个字符串
cin.getline读取一行并丢弃换行符，eg.cin.getline(读入的数组名，要读进的字符数目+1）,被读进的数组里最后总是存放的’\0’
cin.get读取一行但不丢弃换行符，换行符仍然保留在缓冲区中.eg.cin.get(读入的数组名，要读入的字符数目+1）,但保留了换行符，可以使用cin.get()来消化掉这个换行符，所以可以连起来用cin.getline(Arr,Size).get();
cin.get还可也这样用,eg.c=cin.get();在c中使用c=getchar(),同样cout有cout.put(c),等价于c中的putchar(c)
当cin读取到空行或超出了分配的空间，则会设置失效位，将剩余的数据留在缓冲区，并拒绝以后的输入，必须用cin.clear清除失效位，可以使用cin作为判断输入是否正确的判断
11.关于string类的使用9
string类定义在std中
string拼接用+号，可以直接将str1赋值给str2，而在c中，需要使用strcpy(str2,str1),strcat(str2,str1)
string类有长度函数，str.size(),而在c中则使用strlen(str)函数
有一个单独的getline(cin,str)函数，用以将cin的输入读入str中
可以将string类进行比较，eg.str1!=”hello”.但是在c中必须用strcmp(str1,str2)
12.关于c++中的结构体
c++的结构体定义为struct name {…};声明时直接为 name var
而在c中，结构体定义为struct name{…};声明时为struct name var，若要省略struct则要用typedef struct name{…}new_name；声明时为new_name var
struct1可以被赋值给struct2
结构体中可使用位字段eg.strcut a{unsigned int SN:4;}使得SN占4位
13.关于c++中的枚举类型
定义枚举enum name {a,b,c},声明时为name var,var只可以取a或b或c，对应整数值0,1,2，name可以省略，可以直接使用a,b,c这三个enum型
可以更改enum默认取值eg.enum {a=1，b=2}；
枚举值为整型，但是不能自动转换int，不能对枚举型赋予int,但可以是enum1=enum2，最好不要讲枚举相加减，可能导致错误
14.关于c++中的指针
c++中的指针使用基本同c，但可以用new,eg.int p=new int;开辟一个int大小的内存给指针p；若new无法分配内存，则会返回0并且引发bad_alloc异常
用new分配的内存指针需用delete删除,eg.delete p;
将new应用于数组时的方法，eg.int p=new int[10],删除时必须用delete [] p;
数组名视作常量指针，指针加下标会自动解引用,eg.p[1]与(p+1)等价
一般说来cout输出指针会以十六进制输出地址，但对于char指针，则会出书字符串
15.关于c++中延时函数
首先引入，其中有一个CLOCK_PER_SECOND的常量
其次，clock()返回当前系统时间，返回类型为clock_t
因此延时delay定义为clock_t delay=秒数CLOCK_PER_SECOND
16.关于eof
eof表示文件或者输入流结束，在windows中使用ctrl-z+回车模拟，linux中用ctrl-d模拟
检测eof使用cin.eof()或者cin.fail(),后者更加通用，因为遇到eof后,cin将failbit和eofbit都为1，因此必要时用cin.clear清除这两位
cin.get()可以捕捉到eof，所以可以使用cin.get()==EOF作为判断条件
17.关于c++中的字符函数库
字符函数库,其中包含isalpha检测是否为字母，isdigits是否为数字，isspace是否为空字符或其他控制字符，ispunct是否为标点etc,具体参阅说明
18.关于switch语句
switch语句从读取到匹配的分支开始执行，如果没有break则会一直向后执行
19.关于c++中的文件处理
//输出到文件
首先要引入,里面有一个ofstream类，使用时要先声明其变量，打开文件用open函数，关闭用close
eg.
ofstream file;//声明变量
file.open(“filename”);//“filename”可用一个字符串替代
double d;
d>>file;
file.close();
如果文件不存在则创建
//读取文件

里有一个ifstream类，使用时先声明其变量，仍然使用open函数和close函数
eg.
ifstream file;//声明变量
file.open(“filename”)//“filename”可用一个字符串替代
doube d;
file>>d;
file.close();
如果文件不存在将导致输入失败，应当用file.is_open来检测是否打开成功；file.good指出上一次读取操作是否发生了错误或者是否到了文件尾，也可用eof或fail函数具体检测原因
